本程序需要在linux环境下运行 

由于CVE-2017-7494漏洞利用条件:
1. 服务器打开了文件/打印机共享端口445，让其能够在公网上访问

2. 共享文件拥有写入权限

3. 恶意攻击者需猜解Samba服务端共享目录的物理路径

所以本程序也只针对该漏洞，并不提供类似于nmap指纹扫描的功能，另外本程序并没有加载攻击利用模块，仅作版本检测

本程序默认使用匿名登录，若不支持匿名登录，请提供账户（批量模式下不适用，默认匿名登录）
支持CIDR模式扫描 python scan.py -m 192.168.11.0/24或者python scan.py --mass 192.168.11.0/24
支持导入IP扫描   python scan.py -f ./ip.txt 或者python scan.py --file ./ip.txt
支持单个目标扫描 python scan.py -t 192.168.111.136或者python scan.py --target 192.168.111.136
